# ⚡ EV Data Quality & Processing Project

## 📌 Overview
This project focuses on analyzing and improving the quality of Electric Vehicle (EV) datasets. It implements an automated pipeline to detect, clean, and preprocess data by handling missing values, inconsistencies, and duplicates.

The goal is to ensure reliable and accurate data for further analysis, visualization, or machine learning applications.

---

## 🚀 Features
- 🔍 Data Quality Checks
  - Detect missing values
  - Identify duplicate records
  - Validate data types

- 🛠️ Data Preprocessing
  - Data cleaning and formatting
  - Handling inconsistent values
  - Standardization of data

- 🤖 Automated Pipeline
  - End-to-end workflow from raw data to cleaned dataset
  - Modular and reusable design

- 📊 Reporting
  - Summary of data issues
  - Clean dataset output for analysis
